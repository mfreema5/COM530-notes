###Cascading style sheets

```HTML
   <p class="important">  </p>

   <p> </p>
```

```HTML
   p {
   color:red
   }

   p.important {
   font-weight:bold
   }
```
(Does bold red)

```HTML
   p.important {
   font-weight:bold
   color:white
   }
```
(Does bold white)


----

If you specify things like "font" in the `html`, it "breaks the cascade".

----

###Scoring for styling precedence:

| entity   | score | example           |
|----------|-------|-------------------|
| Elements | 1     | `<p>`             |
| Class    | 10    | `<p class="foo">` |
| ID       | 100   | `<p id="bar">`    |
